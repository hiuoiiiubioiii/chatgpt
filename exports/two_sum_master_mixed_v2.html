Two Sum

Master Artifact

Slide 1 — Code Logic

WHAT

Find two distinct numbers in an array that add up to a specific target value, returning their indices.

WHY

This is the foundational problem of "complement search." Instead of mathematically searching for pairs (O(N^2)), we transform the problem to look for the missing half of a pair (O(N)).

SYNTAX

We use std::unordered_map<int, int> in C++ (Hash Map) to store [value -> index] pairs for O(1) amortized lookups.

Slide 2 — Question Statement with Constraints

Raw Problem

Given an array of integers nums and an integer target, return indices of the two numbers such that they add up to target. You may assume that each input would have exactly one solution, and you may not use the same element twice. You can return the answer in any order.

Constraints

  • 2 <= nums.length <= 10^4
  • -10^9 <= nums[i] <= 10^9
  • -10^9 <= target <= 10^9
  • Only one valid answer exists.

Slide 16 — Brute Force Code in C++

class Solution {
 public:
  std::vector<int> TwoSumBruteForce(const std::vector<int>& nums, int target) {
    for (int i = 0; i < static_cast<int>(nums.size()); ++i) {
      for (int j = i + 1; j < static_cast<int>(nums.size()); ++j) {
        if (nums[i] + nums[j] == target) return {i, j};
      }
    }
    return {};
  }
};

Slide 19 — Optimal C++ Code

class Solution {
 public:
  std::vector<int> TwoSum(const std::vector<int>& nums, int target) {
    std::unordered_map<int, int> seen;
    for (int i = 0; i < static_cast<int>(nums.size()); ++i) {
      const int need = target - nums[i];
      if (seen.contains(need)) return {seen[need], i};
      seen[nums[i]] = i;
    }
    return {};
  }
};
